# fullstack-starter

## requirement
- node v12
- yarn
- mongodb



## deployment
```
yarn
# terminal 1
cd apps/api
yarn run dev

# terminal 2
cd app/front
yarn run dev
```


## configuration
Create `.env` in root.
Options:
- `MONGO_URL` - the mongo url
- `MONGO_DB_NAME` - the mongo database name
- `PORT` - (optional) the port to listen (default: 3000)
